# Power BI Dashboard Implementation Guide

This guide provides step-by-step instructions on how to build the final Power BI dashboard using the clean data generated by our Python pipeline.

## 1. Data Import
1. Open **Power BI Desktop**.
2. Click **Get Data** > **Text/CSV** and select `data/retail_final_cleaned.csv`.
3. Click **Load**.
4. Click **Get Data** > **Excel workbook** and select `data/kpi_summary.xlsx` (optional, as the CSV has the full row-level detail, but useful for quick KPI reference tables).
5. If using the KPI Excel file, select all the KPI sheets and click **Load**.

## 2. Data Modeling
Since the Python pipeline already joined the fact table with the product dimension and enriched it with `category_standard` and `standard_price`, you have a robust, flat, analysis-ready dataset in `retail_final_cleaned.csv`. 

Ensure that:
- `transaction_date_parsed` is recognized as a Date type.
- `revenue`, `price`, `quantity`, and `discount` are recognized as Numeric types.

## 3. Recommended Visualizations

### Top-Level KPIs (Card Visuals)
- **Total Revenue:** Sum of `revenue` (Format as Currency ₹)
- **Total Transactions:** Count (Distinct) of `transaction_id`
- **Unique Products Sold:** Count (Distinct) of `product_name`

### Sales by Category (Donut Chart)
- **Legend:** `category_standard`
- **Values:** Sum of `revenue`

### Revenue Trend Over Time (Line Chart)
- **X-axis:** `transaction_date_parsed` (Hierarchy: Year > Quarter > Month)
- **Y-axis:** Sum of `revenue`
- **Insight:** Highlights the peak trading month (August 2025).

### Sales by City (Map or Bar Chart)
- **Location / Axis:** `city`
- **Values:** Sum of `revenue`

### Top Products (Clustered Bar Chart)
- **Y-axis:** `product_name`
- **X-axis:** Sum of `revenue`
- *Tip: Apply a Top 10 filter to keep the chart clean.*

### Channel Performance (Pie Chart)
- **Legend:** `purchase_location` (online vs offline)
- **Values:** Sum of `revenue`

## 4. Interactive Slicers
Add the following slicers to the canvas to allow interactive drill-down:
- `transaction_year` and `transaction_month_name`
- `city`
- `category_standard`
- `payment_method`

## 5. Formatting & Aesthetics
- **Currency:** Format all `revenue` and `price` fields as Currency (₹ / Rs).
- **Theme:** Apply a professional, consistent corporate color theme (e.g., Executive or Blue/Grey tones).
- **Titles:** Give every chart a clear, descriptive title.
- **Tooltips:** Ensure hover tooltips provide precise revenue amounts.

## 6. Saving
Once complete, save your dashboard in the project folder as `Power BI/retail_dashboard.pbix`.
